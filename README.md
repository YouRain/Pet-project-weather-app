Это второй проект в рамках освоения новой профессии Frontend-разработчик.
Данный pet-проект представляет из себя одностраничный сайт, который предоставляет прогноз погоды указанного города. Данные по погоде предоставляет сервис OpenWeatherMap, публичный IP предоставляет серивис ipify API, местоположение по публичному IP предоставляет сервис ipwho, карту предоставляет YandexMap. Проект написан без использования фреймворков, только HTML, CSS и ванильный JavaScript. Сайт адаптирован под все устройства. Стилизация правильно работает в таких браузерах как Chrome, Opera, Firefox по версиям, соответствующим 2022 году.
